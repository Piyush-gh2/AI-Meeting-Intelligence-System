# 🧠 AI Meeting Intelligence System

## 🚀 Overview
An AI-powered system that converts meeting discussions into summaries, action items, and decisions using Multi-Agent AI and RAG.

---

## 💡 Features
- Meeting summary generation
- Action item extraction
- Decision detection
- RAG-based insights
- Streamlit dashboard

---

## ▶️ Run
pip install -r requirements.txt  
streamlit run app.py
